This C# file  is a console application designed to generate a monthly sales report. It reads sales data from a JSON file, filters the records by a user-specified year and month, 
calculates the total turnover in USD by applying currency conversion rates, and saves the results to a text file. The code is structured using helper methods to ensure separation 
of concerns and includes robust error handling for file and JSON operations.

It is include a small JSON file  for testing pourpose  and the result file of running the  file with the sales.json file
